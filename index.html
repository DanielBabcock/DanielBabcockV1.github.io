Med Records Keeper

Link: ➜ Rails API Repo

Link: ➜ AngularJS Front-end Repo

Description: Ever go for a Doctor visit and fill out a huge form only to find yourself a bit fuzzy on details like medications or family history? This app is designed to solve that issue and keep those records in detail, as well as serve the globetrotting types without a family Doctor. This app is built with a Ruby on Rails API using SQlite3 as the back-end and for the front-end it uses AngularJs, Saas, Bootstrap, and HTML/CSS. User login is handled with token-based authentication and task running is handled by Grunt.
